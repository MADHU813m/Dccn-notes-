Data Link Layer

	The data link layer is needed to ensure reliable and efficient data transfer between two directly connected nodes by 	handling framing, physical addressing, flow control, and error detection



Data Link Layer

 Logical Link Control (LLC) 
 Media Access Control (MAC)


Sublayers
Logical Link Control (LLC) Sublayer: 
	This is the upper sublayer that provides a logical link between devices. It handles the flow control, error 	checking, and synchronization of data frames. 

Media Access Control (MAC) Sublayer: 
	This is the lower sublayer that manages how devices access the physical medium. It is responsible for MAC addresses, which are used for unique device 	identification, and for controlling access to the network to prevent 	collisions. 



Protocols
Ethernet (IEEE 802.3): A common protocol for wired local area networks (LANs) that operates at the MAC sublayer. 
Wi-Fi (IEEE 802.11): The standard for wireless LANs. 
Point-to-Point Protocol (PPP): Used to transmit data over serial links, such as those used in dial-up connections. 
High-Level Data Link Control (HDLC): A bit-oriented data link-layer protocol that can be used in point-to-point and multipoint connections. 
Serial Line Internet Protocol (SLIP): An older protocol for transmitting IP packets over serial connections
